# DealCalc Calculator V10 Update

This package only updates the calculator system from the V9 site.

Upload/replace these paths in your GitHub repository root:

- `calculator/`
- `calculators/`
- `calculators.html`
- `assets/js/calculators-v10.js`

Keep the rest of your V9 site files as they are.

## What changed

Each calculator now has its own model, fields, formula explanation, result logic, and interpretation:

- MAO Calculator
- ARV Calculator
- Assignment Fee Calculator
- Double Close Calculator
- Land Flip Calculator
- Rehab Calculator
- ROI Calculator
- Cash Flow Calculator
- Cap Rate Calculator
- Wholesale Deal Calculator

The pages load your existing `/assets/js/main.js` and the new `/assets/js/calculators-v10.js`.
