# DealCalc.io Update Bundle

Upload these files to your GitHub repository `rexezllc-star/dealcalc-website`.

## Recommended steps
1. Download and unzip this bundle.
2. Drag all folders/files into the repo root.
3. Replace existing files when prompted.
4. Open `contact.html` and replace `REPLACE_WITH_YOUR_FORMSPREE_ID` with your real Formspree endpoint.
5. Commit changes to GitHub.
6. Vercel should redeploy automatically.
7. After deploy, submit `https://dealcalc.io/sitemap.xml` again in Google Search Console.
