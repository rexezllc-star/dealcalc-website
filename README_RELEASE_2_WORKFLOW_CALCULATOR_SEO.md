# DealCalc Release 2.0 — Workflow + Calculator Suite 2.0 + SEO Expansion

## What this update implements
- Repositions the site around **AI real estate deal analysis** instead of generic underwriting language.
- Adds AI Analyzer → Calculator flow through a “Continue Your Analysis” section.
- Adds calculator → AI Analyzer flow through “Need help finding the numbers?” import cards.
- Upgrades all core calculators to industry-standard baseline models plus better reporting.
- Upgrades the rental calculator into a **Rental Property Analyzer** with unit count, NOI, cap rate, DSCR, cash-on-cash, break-even occupancy, price per unit, and price per square foot.
- Upgrades the rehab calculator into a **Room-by-Room Rehab Estimator**.
- Adds 20 new SEO article pages and updates the Learn Hub.
- Updates sitemap with calculator and article URLs.
- Adds floating quick actions for Analyze / Calculators / Learn.

## Upload instructions
1. Unzip this package.
2. Upload the contents to the GitHub repo root.
3. Choose **Merge** for folders.
4. Choose **Replace** for matching files.
5. Commit and push to GitHub.
6. Let Vercel redeploy.
