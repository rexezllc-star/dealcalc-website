# DealCalc Article Template

## SEO Metadata
- SEO Title
- Meta Description
- Canonical URL
- OG Title
- OG Description

## Article Structure
- Kicker / Category
- H1
- Lead paragraph
- Search-intent intro
- Practical investor workflow
- Examples
- Cost / comparison tables where relevant
- Common mistakes
- Related calculators
- Related guides
- FAQ
- CTA

## Internal Link Requirements
Each article should link to:
- One primary calculator
- One secondary calculator
- AI Deal Analyzer
- At least three related articles

## Quality Standard
- Search-intent aligned
- Practical and investor-focused
- Clear examples
- FAQ coverage
- No keyword stuffing
- Supports one topical cluster
